This Mod is ONLY compatible with osu!(lazer). I am NOT responsible for damage to devices, so I'll walk you through how to properly use this.
First, download the .DLL file in Releases. (it's named "osu.Mod.MehDrainsHealth2").
Next, place the .DLL file in your rulesets folder. (You can get to your rulesets folder by loading up osu!, go to settings, and click "Open osu! folder".)
Lastly, restart osu! and have fun!
